# Story 5.1: Landing Page Build — Design + Responsivo + CTA

**EPIC:** 5 — Marketing
**Story ID:** 5.1
**Status:** 🆕 Draft
**Priority:** 🔴 MUST (Sem landing page, não tem inbound)
**Assigned to:** @dev
**Depends on:** None (independent)

---

## 📋 Descrição

Construir **landing page web** baseada na especificação copywriting (`landing-page-formulario.md`):

1. **Conteúdo (8 seções):**
   - Hero section (headline + subheadline + CTA)
   - Problema section (3 cenários)
   - Solução section (3 descobertas)
   - Como Funciona section (2 fases)
   - Timeline section (visual 15 dias)
   - Social Proof section (3 depoimentos)
   - CTA Final (call-to-action destacado)
   - FAQ section (4 perguntas)

2. **Design:**
   - Cores: Verde escuro #1B5E20 + branco #FFFFFF
   - Tipografia: Headlines 48-56px bold, body 16-18px regular
   - Layout: Single page, scroll natural, mobile-first
   - Spacing: 80-100px entre seções

3. **Funcionalidade:**
   - Navegação smooth scroll
   - Botões CTA → levam para formulário (Story 1.2)
   - Responsivo: 375px (mobile) até 1920px (desktop)
   - Performance: <2 seg load time

4. **Hosting:**
   - Deploy em Vercel/Netlify (automático com git push)
   - Domain: genialidade.com.br (seu domínio)
   - SSL automático
