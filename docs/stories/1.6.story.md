## 📋 Descrição

Criar **portal web seguro para cliente** acessar:

1. **Autenticação:** Email magic link (sem senha) — clica link no email → acessa portal
2. **Dashboard simples:**
   - Score visual (4 dimensões com cores: vermelho 0-2, amarelo 2-4, verde 4-5)
   - Verdade Núcleo exibida em grande (1-2 frases)
   - Botão "Download PDF" (Story 1.5)
   - Botão "Preparar-se para sessão" (lista o que levar)
   - Data/hora da sessão 1:1 agendada
3. **Timing:** Disponível imediatamente após aprovação (dia 6)
4. **Privado:** Cliente vê só seus dados, autenticação segura
5. **Mobile:** Responsivo para celular (cliente consulta no caminho para sessão)

---

## ✅ Acceptance Criteria (10 pontos)

- [ ] 1. **Magic link auth:** Email contém link único (token expiração 24h), clica → autenticado
- [ ] 2. **Session management:** Cliente permanece autenticado por 7 dias (cookie seguro ou JWT)
- [ ] 3. **Dashboard renderiza:** Página carrega score do banco (Story 1.1), sem delay
- [ ] 4. **Score visual:** 4 dimensões mostradas com cores (0-2 vermelho, 2-4 amarelo, 4-5 verde)
- [ ] 5. **Verdade Núcleo destaque:** Exibida em tipografia grande (32-48px), legível, específica
- [ ] 6. **Download PDF:** Botão funciona, baixa PDF gerado (Story 1.5)
- [ ] 7. **Info Sessão:** Data, hora, link Zoom, "o que levar" (checklist para cliente)
- [ ] 8. **Privacidade:** Cliente vê só seus dados (RLS enforce, teste com 2 clientes)
- [ ] 9. **Mobile responsive:** Teste em viewport 375px, tudo acessível sem scroll horizontal
- [ ] 10. **Segurança:** Magic link não pode ser reutilizado, token armazenado seguro (bcrypt hash)

## 📌 Notas

- **Magic link é mais simples que OAuth:** Sem criar senha, cliente entra direto
- **7 dias de session:** Suficiente para cliente preparar-se antes da sessão (dia 6) + acesso pós-sessão (dias 8-15) no Caderno Inteligente (Story 3.2)
- **Minimalista é feature:** Não adicione chat, messaging, etc nessa story — mantem focado
- **Branding:** Consistente com landing page (colors, fonts) para sensação de coesão
- **Preparação sessão:** Checklist simples (ex: trazer 3 últimos 3 meses P&L, 2 maiores problemas da semana, 1 coisa que quer alcançar em 90 dias)
