## 📋 Descrição

Gerar **PDF do Diagnóstico Formal** automaticamente após aprovação Fase 1:

1. **Trigger:** Cliente aprovado (score ≥12 AND maturidade ≥4)
2. **Conteúdo PDF (5 páginas):**
   - **P1:** Capa (Logo, Nome Cliente, Data, "Diagnóstico Neural Estruturado")
   - **P2:** Resumo Executivo (Verdade Núcleo em 1-2 frases)
   - **P3:** Scores detalhados (4 dimensões com gráficos simples)
   - **P4:** Hipótese estruturada (3-5 frases + padrões detectados)
   - **P5:** Próximos Passos (data da sessão 1:1, o que levar, etc)
3. **Estilo:** Elegante, profissional, brandizado com cores genialidade (verde escuro + branco)
4. **Entrega:** Email automático com PDF anexado, +  acesso portal
5. **Timing:** Gerado em ≤30 seg após aprovação

---

## ✅ Acceptance Criteria (10 pontos)

- [ ] 1. **PDF renderiza:** Gerador PDF (pdfkit/puppeteer/similar) renderiza 5 páginas corretamente
- [ ] 2. **P1 Capa:** Logo, cliente nome, data, título legível
- [ ] 3. **P2 Resumo Executivo:** Verdade Núcleo em linguagem clara (não jargão)
- [ ] 4. **P3 Scores:** 4 dimensões com valores numéricos + gráfico de barras simples (não complexo)
- [ ] 5. **P4 Hipótese:** Padrões detectados de respostas, específico ao cliente (não genérico)
- [ ] 6. **P5 Próximos Passos:** Data/hora sessão, agenda, o que levar, contato
- [ ] 7. **Branding:** Cores genialidade (verde escuro #1B5E20 + branco #FFFFFF), fonte legível (sans-serif)
- [ ] 8. **Email anexado:** Email Story 1.4 inclui PDF como attachment, aparece no cliente email
- [ ] 9. **Persistência:** PDF salvo em storage (Supabase storage ou S3), link rastreável
- [ ] 10. **Performance:** Geração leva <30 segundos, sem bloquear fluxo

## 📌 Notas

- **PDF não é comunicação primária** — é comprovante. Comunicação real é na sessão (Story 2.1)
- **Verdade Núcleo deve ser específica:** "Sua dispersão de oferta está drenando 35% da sua margem" (não "seu negócio tem problemas")
- **Gráficos devem ser simples:** Barras 4 cores diferentes, sem complexidade. Cliente já está absorvendo muita informação
- **5 páginas é máximo:** Mais virava relatório maçante, menos não comunica rigor
- **Branding é importante:** PDF elegante reforça que você é premium (não genérico)
