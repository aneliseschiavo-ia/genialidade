## 📋 Descrição

Gerar **documento formal (PDF) com 7 páginas** que:

1. **Captura de sessão:** Leva dados coletados em Story 3.1 (Guia Sessão)
2. **Estrutura clara:** 7 páginas específicas (Verdade Núcleo, Premissa Corrigida, Cortes, Decisões, Plano 7D, Indicador, Check-in)
3. **Entregável final Fase 2:** Cliente recebe dia 8 (após sessão)
4. **Validação cliente:** Documento que cliente assina (sim, está de acordo com o que conversamos)
5. **Ativo próximos 7D:** Cliente executa Plano 7D conforme documento
6. **Rastreável:** Versão final armazenada (audit trail)

---

## ✅ Acceptance Criteria (10 pontos)

- [ ] 1. **P1 Capa:** Titulo "Blueprint 7D de Validação", nome cliente, data sessão, assinatura cliente (digital ou campo texto)
- [ ] 2. **P2 Verdade Núcleo:** 1-2 frases descritivas (cópia da Seção 1 Caderno Inteligente)
- [ ] 3. **P3 Premissa Corrigida:** Antes (da Fase 1) vs Depois (refinada na sessão) — comparação clara
- [ ] 4. **P4 3 Cortes:** Tabela (O quê, Por quê, Impacto esperado) — específico, não genérico
- [ ] 5. **P5 3 Decisões:** Tabela (Decisão, Métrica, Responsável, Timeline) — cada decisão tem "dono" e prazo
- [ ] 6. **P6 Plano 7D:** 7 dias estruturados (Dia 1-7: X a fazer, Y a rastrear, Z a validar)
- [ ] 7. **P7 Indicador + Check-in:** Métrica crítica + template 5 campos para cliente preencher diariamente/semanalmente
- [ ] 8. **Branding:** Cores genialidade, tipografia consistente, layout profissional
- [ ] 9. **Assinatura validação:** Campo "Cliente concorda" (checkbox ou assinatura digital) — prova que cliente validou conteúdo
- [ ] 10. **Persistência:** Armazenado em Storage, link rastreável, versionado (se cliente pedir revisão dia 10, guarda v1 e v2)

## 📌 Notas

- **Blueprint NÃO é teoria:** Cada linha deve ser executável (não "melhorar comunicação", mas "reunião status 2x/semana → 1x/semana, economiza 8h/semana")
- **Plano 7D é exigente:** Cliente não deveria poder fazer tudo em 7 dias. Deve ser focused (3 ações máximo por dia)
- **Indicador é validação:** Se cliente não consegue medir resultado em 7 dias, indicador foi mal escolhido
- **Assinatura cliente:** Prova que cliente validou e está comprometido (legal + psicológico)
- **Versionamento important:** Se cliente quer ajuste dia 10 (semana in progress), guarda v1 original + v2 adjusted
