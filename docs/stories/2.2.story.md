# Story 2.2: Agendamento Sessão 1:1 — Calendário + Zoom/Meet

**EPIC:** 2 — Processo Fase 1→2
**Story ID:** 2.2
**Status:** 🆕 Draft
**Priority:** 🔴 MUST (Cliente precisa agendar antes de Fase 2)
**Assigned to:** @dev
**Depends on:** Story 2.1 (Aprovação), Story 1.6 (Portal Cliente)

---

## 📋 Descrição

Criar **sistema de agendamento de sessão 1:1** que:

1. **Pré-requisito:** Cliente aprovado (Story 2.1) recebe magic link (Story 1.6)
2. **UI agendamento no portal:**
   - Cliente vê **sua disponibilidade (anote manualmente ou integre Calendly)**
   - Clica em horário disponível
   - Confirma dados (email, telefone para Zoom invite)
3. **Integração Zoom/Google Meet:**
   - Gera automaticamente link Zoom para sessão
   - Link armazenado no banco (tabela `sessoes`)
4. **Email confirmação:**
   - Cliente recebe confirmação (Story 2.3)
   - Você recebe notificação (cliente agendou, data/hora, link Zoom)
5. **Timing:** Disponível para agendar dias 6-7 (após aprovação), Fase 2 sessão é até dia 8
6. **Graceful reschedule:** Se cliente precisa reagendar, pode fazer até 24h antes

## 📌 Notas

- **Calendly optional:** Se quiser simplicidade, pode configurar manualmente sem Calendly (você diz "slots terça 14-15:30, quarta 10-11:30" e clientes escolhem)
- **Zoom gerado automaticamente:** Cliente não precisa fazer nada, link vem no email
- **Reschedule graceful:** Permitir até 24h antes, depois disso precisa email direto pra você
- **Timezone:** Brasil é UTC-3, tenha cuidado se você tiver clientes em fusos diferentes
- **90 min minimum:** Sessão 1:1 é 75-90 min, deixa 15 min buffer entre slots
