## 📋 Descrição

Criar schema de banco de dados no Supabase para armazenar:
- **Clientes** (email, nome, data inscrição)
- **Questões** (28 perguntas pré-definidas do formulário diagnóstico)
- **Respostas** (respostas do cliente por questão)
- **Scores** (cálculo automático: 4 dimensões × 5 pontos = 0-20)
- **Decisões** (Score, Hipótese, Aprovado/Rejeitado, data análise)

Integrar Supabase com ambiente Node.js/Next.js do projeto.
