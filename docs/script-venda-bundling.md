## 3. SCRIPT #1: UPSELL 90D (Após Aprovação Fase 1)

**Quando usar:** Dia 6–7 (quando envia hipótese preliminar + aprovação para Fase 2)

**Contexto:** Cliente foi aprovado. Motivação é alta. Momento de oferecer acompanhamento 90D.

### **Email/Mensagem de Aprovação + Upsell**

```
Oi [Nome],

Finalizei a análise do seu formulário. Boas notícias.

SCORE:
Desalinhamento: [X]/5
Ruído: [X]/5
Vazamento: [X]/5
Maturidade: [X]/5
TOTAL: [X]/20 ✅ APROVADO

HIPÓTESE PRELIMINAR:
[Sua frase única]

Isso significa: você passou no corte. Vamos pra Fase 2.

---

A gente vai sair da sessão (dia [data]) com 3 coisas claras:
1) Verdade Núcleo (qual é realmente o problema)
2) 3 Cortes (o que parar de fazer)
3) 3 Decisões + Plano 7D (testar a solução)

Você vai ter 7 dias pra validar se a premissa corrigida funciona.

No dia 7, você vai saber se a direção está certa.

---

AGORA AQUI TEM UMA COISA IMPORTANTE:

Se a premissa funcionar (e vai funcionar — score de maturidade seu é sólido),
a próxima pergunta é: "Como faço isso escalar sem voltar pra padrão antigo?"

Porque validar em 7 dias é uma coisa. Manter funcionando 90 dias é outra.

Tenho um pacote de 90 dias que é EXATAMENTE pra isso:
- Check-ins semanais (você me passa dados, eu analiso)
- Sessões mensais (90 min juntos pra ajustar rota)
- Mensuração de ROI
- Escalação estruturada

Total: 13 semanas de suporte estruturado.
Investimento: R$ 1.500/mês (R$ 4.500 total)

O legal é: você já vai validar que a premissa funciona em 7 dias.
Aí você entra no 90D sabendo que está no caminho certo.

Quer conversa de 15 min sobre isso antes da Fase 2?
Ou prefere primeiro rodar Fase 2 + Blueprint, testar os 7 dias,
e aí a gente conversa sobre 90D?

Ambas opções fazem sentido.

[Seu nome]
```

**Notas:**
- Não é agressivo (oferece flexibilidade)
- Valida a premissa de valor (escala requer estrutura)
- Oferece duas opções (reduz objeção)

## 4. SCRIPT #2: UPSELL FINAL 90D (Após Entrega Blueprint)

**Quando usar:** Dia 15 (quando entrega diagnóstico + blueprint 7D)

**Contexto:** Cliente tem tudo em mãos. Motivação é altíssima. Próximo passo: execução 7D → depois 90D.

### **Email Pós-Blueprint + Call de 15 min**

```
Oi [Nome],

Diagnóstico + Blueprint chegaram!

Abra agora: [Link Google Drive]

Você tem tudo ali:
- Diagnóstico completo (PDF)
- Blueprint 7D (para executar Dias 16–22)
- Caderno inteligente compartilhado (você preenche enquanto executa)

PRÓXIMOS 7 DIAS:
- Dia 1–7: Execute o Blueprint conforme planejado
- Dia 7: Preencha o check-in final (indicador crítico)
- Eu recebo o resultado

DEPOIS DOS 7 DIAS:
Você vai saber se a premissa corrigida funciona.

---

AGORA, VAI UMA REFLEXÃO:

Quando você validar que funciona (e vai funcionar), vem a parte difícil:
Como manter isso funcionando? Como isso sustenta crescimento 90 dias?

Porque 7 dias de teste é uma coisa.
Mas 90 dias de execução contínua, ajustes, escalação — é outra.

Eu tenho um programa de acompanhamento de 90 dias que faz exatamente isso:
- Check-ins semanais (dados estruturados)
- Sessões mensais (ajuste de rota + escalação)
- Mensuração de ROI contínua

Você não fica sozinho nessa jornada.

Investimento: R$ 1.500/mês (primeiros 4 clientes) = R$ 4.500 total

TOTAL COMBO (diagnóstico + 90D): R$ 7.000
(Depois vira R$ 4.800 diagnóstico + R$ 2.500/mês acompanhamento = mais caro)

---

QUER CONVERSA RÁPIDA (15 min)?

Sem compromisso. Só pra você entender como funciona o 90D,
e decidir se quer fazer ou só rodar diagnóstico + 7D sozinho.

Você tem tempo [opções de horário]?

[Seu nome]
```

**Timing:** Envie dia 15, quando entrega blueprint. Ofereça call dia 18–20.

## 5. BUNDLING OPTIONS

### **OPÇÃO A: MVP + 90D (Recomendado)**

```
Diagnóstico Neural + Blueprint 7D: R$ 2.500
+ Acompanhamento 90D: R$ 4.500 (R$ 1.500/mês × 3 meses)
────────────────────────────────
TOTAL: R$ 7.000 (combo MVP)

Preço depois: R$ 4.800 + R$ 2.500/mês = mais caro
```

**Quando oferecer:**
- Dia 15 (entrega blueprint) — melhor timing
- Ou Dia 6–7 (aprovação Fase 1) — mais agressivo

**Argumento:**
"Você já validou em 7 dias que a premissa está certa.
90 dias é pra garantir que escala sem voltar pra padrão antigo.
Combo sai mais barato que comprar separado depois."

---

### **OPÇÃO B: Só MVP (Sem Acompanhamento)**

```
Diagnóstico Neural + Blueprint 7D: R$ 2.500

Cliente executa sozinho os 7 dias.
Sem acompanhamento semanal.
```

**Quando cliente escolhe:**
- Prefere fazer sozinho
- Orçamento limitado agora
- Quer testar antes de comprometer 90D

**Comunicação:**
"Tá bom, sem problema. Você executa diagnóstico + 7D sozinho.
Se depois de 7 dias quiser começar acompanhamento, é só chamar.
Preço vai ser normal (não MVP), mas a gente negocia."

---

### **OPÇÃO C: Só 90D (Cliente que já tem diagnóstico)**

```
Acompanhamento 90D (sem diagnóstico prévio): R$ 1.500/mês

Cliente que já sabe qual é o problema, quer estrutura de execução.
```

**Quando oferece:**
- Cliente referido que já conhece seu método
- Cliente que fez diagnóstico com você 6 meses atrás

---

### **OPÇÃO D: Diagnóstico + 90D + Próximo Pacote**

```
Diagnóstico: R$ 2.500
Acompanhamento 90D: R$ 4.500
+ Pacote Escalação (próximos 90D): R$ 7.500–10.000
────────────────────────────────
TOTAL: R$ 14.000–16.500 (contrato anual)

Desconto de 10%: R$ 12.600–14.850
```

**Quando oferecer:**
- Cliente é grande (R$ 50k+/mês faturamento)
- Cliente quer comprometimento anual
- Dia 90 (fim do primeiro acompanhamento)

### **"Por que R$ 7.000 combo é melhor que R$ 2.500 + depois contratar 90D?"**

**Resposta:**
```
"Ótima pergunta. Aqui é a matemática:

Opção A (Combo agora):
- Diagnóstico MVP: R$ 2.500
- 90D MVP: R$ 4.500
- Total: R$ 7.000

Opção B (Depois):
- Diagnóstico MVP: R$ 2.500
- Diagnóstico Regular (depois): R$ 4.800 (preço sobe)
- 90D Regular: R$ 7.500 (preço sobe)
- Total: R$ 14.800

Diferença: R$ 7.800

Além disso: No combo você SABE que vai fazer 90D,
então tem momentum/continuidade.
Fazer só diagnóstico, testar 7 dias, e aí depois voltar a fechar 90D —
muita gente perde motivação.

Combo é investimento + comprometimento.
Você valida diagnóstico AND escala estruturado.

Faz sentido?"
```

## RESUMO EXECUTIVO

**Seu Rol no Processo de Venda:**

1. **Marketing + Landing Page** (conteúdo + design) — Gerar curiosidade, dirigir ao formulário
2. **Formulário Fase 1** (prospect preenche) — Pitch silencioso + filtro + qualificação
3. **Análise + Aprovação** (você) — Calcular score, enviar hipótese, oferecer Fase 2
4. **Upsell 90D #1** (você, email) — Logo após aprovação (timing: dia 6–7)
5. **Sessão Fase 2** (você, 75–90 min ao vivo) — Diagnóstico + blueprint
6. **Upsell 90D #2** (você, email + call 15min) — Logo após entregar blueprint (timing: dia 15, momento quente)
7. **Check-ins semanais** (você, se 90D contratado) — Manter cliente engajado
8. **Sessões mensais** (você) — Ajustar rota
9. **Relatório ROI final + Renovação** (você) — Día 90, próximos passos
