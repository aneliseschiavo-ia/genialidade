## EXECUTIVE SUMMARY

### O que é?

Um pacote de acompanhamento semanal de 90 dias para clientes que completaram o diagnóstico + blueprint 7D. Objetivo: garantir que a premissa validada sustenta escala, mensurar ROI, e ajustar conforme necessário.

### Visão Estratégica

```
Diagnóstico + Blueprint 7D (R$ 2.500)
→ Identifica premissa errada + testa solução em 7 dias
→ Validação: "Funciona?"

ENTÃO VEM: Acompanhamento 90D (R$ 1.500–2.500/mês)
→ "Mantém funcionando enquanto escala"
→ Monitoramento semanal + ajustes de rota
→ Mensuração de ROI real
```

### Proposta ao Cliente

```
"Agora que você conhece a premissa corrigida,
vamos garantir que ela sustenta crescimento escalável.
90 dias de check-ins semanais para validação contínua."
```

### Viabilidade

- **Duração:** 90 dias (13 semanas)
- **Investimento:** R$ 1.500–2.500/mês (R$ 4.500–7.500 total)
- **Formato:** Check-in semanal assíncrono + 1 sessão ao vivo/mês
- **Objetivo:** Escalação validada + mensuração de ROI
- **Upsell Window:** Ofereça no dia 15 (quando entrega blueprint)

## 1. VISÃO & OBJETIVOS

### 1.1 Visão do Produto

Transformar a validação de 7 dias em escala sustentável de 90 dias — através de monitoramento estruturado, ajustes iterativos, e mensuração de ROI real.

### 1.2 Objetivos

| Objetivo | Métrica | Target |
|----------|---------|--------|
| Implementação Executada | % ações blueprint executadas | ≥90% |
| Indicador Evolui | Crescimento do indicador crítico | +20% em 90 dias |
| ROI Documentado | Receita gerada / Investimento | ≥3x em 90 dias |
| Escalação Iniciada | Cliente replicou decisão principal | 1+ vezes |
| Satisfação | NPS no check-in final | ≥8/10 |

### 1.3 Diferenciação vs Coaching Genérico

**NÃO é:** Coaching aberto onde você "pergunta como está"

**É:** Monitoramento estruturado onde você valida se premissa continua sustentando realidade conforme cliente escala.

```
Coaching genérico: "Como você se sente sobre isso?"
Acompanhamento neural: "Seu indicador crítico evoluiu? Por quê? Qual ajuste fazer?"
```

## 3. ESTRUTURA — 13 SEMANAS

### 3.1 Visão Geral da Sequência

```
SEMANA 1–2: Execução Blueprint (dias 16–22)
↓
SEMANA 3: Check-in Confirmação (dia 7 blueprint + validação)
↓
SEMANA 4–13: Acompanhamento Estruturado
  - Check-ins semanais (assíncrono + dados)
  - Sessão mensal ao vivo (ajustes + escalação)
  - Mensuração de ROI contínua
```

### 3.2 SEMANA 1–2: Execução Blueprint (Preparação)

**Objetivo:** Cliente executa 7 dias de blueprint conforme planejado

**Seu Papel:**
- Confirmar que cliente começou
- Estar disponível para dúvidas (não proativo, responde se pedir)
- Rastrear se está no calendário

**Check-in Mínimo:** Mensagem no dia 1: "Começou? Qualquer dúvida, é aqui"

**Saída:** Cliente completou dia 7, conhece resultado do indicador

### 3.3 SEMANA 3: Check-in Confirmação (Validação Dia 7)

**Objetivo:** Validar que premissa sustentou realidade (indicador evoluiu)

**Seu Papel:**
- Receber resultado do indicador crítico
- Análise: "Sustentou?"
- Decisão: Continuar ou ajustar premissa

**Formato:** Email estruturado do cliente + sua análise

**Possíveis Resultados:**

| Resultado | Próximo Passo |
|-----------|---------------|
| **Indicador evoluiu (sustentou)** | ✅ Continue escalando com confiança |
| **Indicador estagnou** | ⚠️ Ajuste uma variável (qual ação foi incompleta?) |
| **Indicador piorou** | 🔴 Premissa ainda não está certa (análise mais profunda) |

**Comunicação ao Cliente:**

Se sustentou:
```
"Ótimo. Sua premissa corrigida sustentou a realidade. Agora o desafio é manter isso
funcionando enquanto você escala. Vamos usar as 12 semanas próximas para isso."
```

Se ajuste necessário:
```
"O indicador ainda não evoluiu como esperado. Qual das 3 decisões você sente que
ficou incompleta? Vamos ajustar ali."
```

### 3.4 SEMANA 4–13: Acompanhamento Estruturado (12 semanas)

**Formato Principal:** Check-in semanal assíncrono + 1 sessão ao vivo/mês

#### **3.4.1 Check-in Semanal (Assíncrono)**

**Frequência:** Todo domingo ou segunda-feira

**O que Cliente Envia (2 min para preencher):**
```
[ ] Executei as 3 ações principais esta semana? Sim / Parcialmente / Não
[ ] Qual % do tempo (aprox) dediquei à oferta principal? ____%
[ ] Indicador crítico (números): Semana passada ___, Esta semana ___
[ ] Um ajuste que preciso fazer: ___
[ ] Uma vitória que tive: ___
```

**O que Você Faz (5 min análise):**
1. Lê resposta
2. Detecta desvios (% tempo caiu? Indicador regrediu?)
3. Identifica padrão (mesma dificuldade 2 semanas seguidas?)
4. Responde com foco: "Vejo que você caiu em X. Vamos ajustar isso na sessão."

**Resposta Sua (Email/Mensagem):**
```
"Vejo que você dedicou 40% do tempo (meta era 60%).
Isso coincidiu com indicador regredir 5%.
Não é ruim — é dado. Na sessão essa semana, vamos entender o que concorreu com seu tempo.

Sua próxima ação: Rastreie 30 min qual atividade competiu com foco principal."
```

**Timing:** Responder em até 24h

#### **3.4.2 Sessão Mensal ao Vivo (1/mês, 45 min)**

**Frequência:** Semana 1, 5, 9, 13 (ou adaptado conforme fluxo)

**Estrutura (45 minutos):**

1. **Validação de Dados (10 min)**
   - Revisar check-ins do mês
   - Detectar padrões
   - Confirmar números do indicador

2. **Identificação de Bloqueio (15 min)**
   - Qual ação ficou incompleta? Por quê?
   - Qual crença emergente está competindo com premissa? (ex: "Talvez eu deveria...")
   - Qual decisão não está sendo honrada na prática?

3. **Ajuste de Rota (10 min)**
   - Se necessário, ajustar ação
   - Se necessário, revisar decisão (talvez premissa precisa evolução?)
   - Se necessário, mudar indicador (capturar melhor a realidade)

4. **Escalação Planejada (10 min)**
   - Indicador está evoluindo? → Próximo passo é escalar
   - Como você replica a decisão principal?
   - Qual novo investimento faz sentido?

**Saída da Sessão:**
- Plano semanal para próximas 3–4 semanas
- Ajustes confirmados
- Indicador revisado (se necessário)

#### **3.4.3 Padrão Esperado das 13 Semanas**

```
SEMANAS 1–3: Validação
- Indicador ainda pequeno/novo
- Foco: Execute o que foi planejado
- Ajustes: Pequenos (tática)
- Frequência check-in: Semanal

SEMANAS 4–8: Consolidação
- Indicador começou evoluir
- Foco: Expandir o que funciona
- Ajustes: Médios (premissa está certa, tática muda)
- Frequência check-in: Semanal + sessão mensal

SEMANAS 9–13: Escalação
- Indicador está forte
- Foco: Replicar em nova área
- Ajustes: Grandes (como escalar sem perder premissa?)
- Frequência check-in: Semanal + sessão mensal
```
